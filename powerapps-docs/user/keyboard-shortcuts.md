# <a name="use-keyboard-shortcuts-in-powerapps"></a>PowerApps でキーボードショートカットを使用する

キーボードショートカットでは、マウスを使用したり、指でタップしたりする代わりに、一般的なタスクを実行する別の方法が提供されます。 インターフェイスをシームレスに移動するのに役立ちます。 次のキーボードショートカットは、アプリ間で適用されます。 

> [!NOTE]
> ここで説明するキーボードショートカットは、米国キーボードレイアウトを示しています。 他のキーボードレイアウトのキーは、US キーボードのキーと正確に対応しない場合があります。

## <a name="navigating-submenus"></a>ナビゲーション (サブメニューを)
スクリーンリーダーを使用する場合は、次の手順に従って、ナビゲーションリンク上にマウスポインターを置いたときに表示されるすべてのサブメニューにアクセスします。

1. 標準リンクナビゲーションを使用してメニューオプション間を移動します。
2. 通常どおり Enter キーまたはスペースキーを選択して、リンクをアクティブにします。 サブメニューの一覧が表示されます。
3. これらのサブメニューオプションにアクセスするには、補助技術 () ショートカットキーを使用して [次へ] ボタンに移動します。 たとえば、NVDA スクリーンリーダーを使用する場合は、"b" ショートカットキー (NVDA + b) を使用します。 すべてのオプションを移動するには、キーを繰り返し選択します。

> [!IMPORTANT]
> ユーザーインターフェイス (UI) のいくつかの領域には、メニュー用に自動折りたたみ機能が組み込まれています。 これには、ナビゲーションバーとフォームナビゲーションが含まれます。 どちらも選択したときに展開されますが、非アクティブな状態が30秒になると項目は折りたたまれます。

## <a name="navigating-through-stages-of-a-process"></a>プロセスの段階間の移動
潜在顧客などのレコードの場合、このセクションでは、営業プロセス内のさまざまな段階を移動して潜在顧客を限定するためのキーボードショートカット、およびそれらのステージで選択する方法を示しています。 たとえば、[分類] ステージから [閉じる] 段階に簡単に移動する場合は、次のキーボード操作を試してください。
- ステージを選択したら、[下へ] (↓) 矢印を選択してフィールドを表示します。
- ステージが折りたたまれている場合は、[Enter] を選択して展開します。

  > [!NOTE]
  > プロセス名は、上部ではなく、ページの下部にあるアイコンです。

- **Windows のキーボードショートカット**:Windows オペレーティングシステムを使用している場合は、次のリンクを選択し、ドロップダウンリストからバージョン番号を選択します。 Windows 環境に適用可能なすべてのショートカットを表示できます。 「 [Windows ショートカットキー](https://support.microsoft.com/en-us/help/12445/windows-keyboard-shortcuts)」を参照してください。 
- **Mac でのキーボードショートカット**:Mac オペレーティングシステムを使用している場合は、次のリンクを選択して、Mac 環境に適用できるすべてのショートカットを表示します。 「 [Mac のキーボードショートカット](https://support.apple.com/en-us/HT201236)」を参照してください。
